# Thiruvarutpayan


|||
|---|:-------------|
|0|**காப்பு**<br/> நற்குஞ்சரக் கன்று நண்ணில் கலைஞானம் <br/>கற்குஞ் சரக்கன்று காண். |


1. பதிமுது நிலை

|||
|---|:-------------|
|1   |**கடவுளின் இயல்பு**<br/>அகர உயிர்போல் அறிவாகி எங்கும்<br/>நிகரில் இறை நிற்கும் நிறைந்து.|
|2  | **சிவசத்தியின் தன்மை**<br/>தன் நிலைமை மன் உயிர்கள் சாரத் தரும்சத்தி<br/>பின்னம் இலான் எங்கள் பிரான்.|
|3  |**சிவத்தின் பெருமை**<br/> பெருமைக்கும் நுண்மைக்கும் பேர்அருட்கும் பேற்றின்<br/>அருமைக்கும் ஒப்புஇன்மை யான்.|
|4  |**சிவத்தின் அருட் செயல்கள்**<br/>ஆக்கிஎவையும் அளித்து ஆசுடன் அடங்கப்<br/>போக்கு அவன் போகாப் புகல்.|
|5  |**அருட்செயல்களை நடத்தும் திருமேனிகள்**<br/>அருவம் உருவம் அறிஞர்க்கு அறிவாம்<br/>உருவம் உடையான் உளன்.|
|6  |**அத்திருமேனிகளைத் தோற்றுவிப்பவர் அவரே**<br/>பல்ஆர் உயிர் உணரும் பான்மைஎன மேல்ஒருவன்<br/>இல்லாதான் எங்கள் இறை.|
|7  |**அருட்செயல்களின் பயன் ஆன்மாவில் சிவம் பிரகாசித்தல்**<br/>ஆனா அறிவாய் அகலான் அடியவர்க்கு<br/>வான்நாடர் காணாத மன்.|
|8  |**சிவத்தின் வியாபகம்**<br/>எங்கும் எவையும் எரி உறு நீர்போல் ஏகம்<br/>தங்கும்அவன் தானே தனி.|
|9  |**திருவருளை நுகரும் வழி**<br/>நலம்இலன் நண்ணார்க்கு நண்ணினர்க்கு நல்லன்<br/>சலம்இலன் பேர் சங்கரன்.|
|10  |**அவ்வழியில் ஆன்மாவைச் செலுத்துதல்**<br/>உன்னும்உளது ஐயம்இலது உணர்வாய் ஓவாது<br/>மன்னுபவம் தீர்க்கும் மருந்து.|

2.  உயிரவை நிலை

|||
|---|:-------------|
|11   |**ஆன்மாக்கள் எண்ணில்லாதன**<br/> பிறந்தநாள் மேலும் பிறக்குநாள் போலும்<br/> துறந்தோர் துறப்போர் தொகை.|
|12  | **மூவகை ஆன்மாக்கள்**<br/> திரிமலத்தார் ஒன்றுஅதனில் சென்றார்கள் அன்றி <br/> ஒரு மலத்தாராயும் உளர்.|
|13  |**மூவகை ஆன்மாக்களின் வேறுபாடு**<br/> மூன்றுதிறத்து உள்ளாரும் மூலமலத்து உள்ளார்கள் <br/> தோன்றலர் தொத்துஉள்ளார் துணை.|
|14  |**ஆன்மா வலியற்றது**<br/>கண்டவற்றை நாளும் கனவில் கலங்கியிடும் <br/>திண்திறலுக்கு என்னோ செயல்?.|
|15  |**ஆன்மா அறிவிக்க அறிவது**<br/> பொறியின்றி ஒன்றும் புணராத புந்திக்கு <br/> அறிவுஎன்ற பேர்நன்று அற.|
|16  |**ஆன்மா இயல்பாக அறிவுப்பொருள்**<br/>ஒளியும் இருளும் உலகும் அலர்கண் <br/> தெளிவுஇல் எனில்என் செய.|
|17  |**ஆன்மா சதசத்து**<br/> சத்துஅசத்தைச் சாராது அசத்துஅறியாது அங்கண்இவை<br/> உற்ற சதசத்தாம் உயிர்.|
|18  |**ஆன்மா தன்வயமில்லாதது**<br/> இருளில் இருளாகி எல்இடத்தில் எல்ஆம் <br/> பொருள்கள் இலதோ புவி.|
|19  |**ஆன்மா சிவத்தைக் காணாமைக்குக் காரணம்**<br/> ஊமன்கண் போல ஒளியும்  மிகஇருளே<br/> ஆம்மன் கண் காணாதவை.|
|20  |**ஆன்மா அருளைப் பெற முயலல் வேண்டும்**<br/> அன்றளவும் ஆற்றும் உயிர்அந்தோ அருள்தெரிவது<br/> என்று? அளவொன்றுஇல்லா இடர்.|

3.  இருண்மல நிலை

|||
|---|:-------------|
|21   |**முப்பொருளும் இன்ப துன்பமும் உள்ள பொருள்கள்**<br/> துன்றும் பவத்துயரும் இன்பும் துணைப்பொருளும் <br/> இன்றுஎன்பது எவ்வாறும் இல்.|
|22   |**ஆணவத்தின் உண்மை அனுமானத்தால் துணியப்படும்** <br/>இருளானது அன்றி இலதுஎவையும் ஏகப்<br/>பொருளாகி நிற்கும் பொருள் .|
|23   |**ஆணவம் இருளிற் கொடியது** <br/>ஒருபொருளும் காட்டாது இருள்உருவம் காட்டும் <br/> இருபொருளும் காட்டாது இது.|
|24   |**ஆணவம் அநாதியாய் ஆன்மாவோடு உள்ளது**<br/> அன்றுஅளவி உள்ளொளியோடு ஆவிஇடை அடங்கி <br/> இன்றுஅளவும் நின்றது இருள்.|
|25   |**ஆணவம் அநந்த சத்திகளை உடையது**<br/> பலரைப்புணர்ந்தும் இருள்பாவைக்கு உண்டு என்றும் <br/>கணவர்க்கும் தோன்றாத கற்பு.|
|26   |**ஆணவம் மெய்யுணர்வைத் தடுப்பது**<br/> பன்மொழிகள்என்? உணரும் பான்மை தெரியாத <br/> தன்மை இருளார் தந்தது.|
|27   |**ஆணவம் ஆன்மாவின் குணமன்று**<br/>இருள்இன்றேல் துன்புஏன்? உயிர்இயல்பேல் போக்கும் <br/>பொருள்உண்டேல் ஒன்றாகப் போம்.|
|28   |**ஆணவம் அநாதி**<br/> ஆசுஆதியேல் அணைவ காரணம்என்? முத்திநிலை <br/> பேசாது அகவும் பிணி.|
|29   |**ஆணவத்தைக் கெடுக்கும் ஒரு கருவி மாயை**<br/> ஒன்று மிகினும் ஒளிகவராதேல் உள்ளம் <br/> என்றும் அகலாது இருள்.|
|30   |**மாயை கன்மங்களின் இயல்பு**<br/> விடிவுஆம் அளவும் விளக்குஅனைய மாயை <br/> வடிவுஆதி கன்மத்து வந்து.|

4.  அருளது நிலை

|||
|---|:-------------|
|31   |**அருளின் பெருமை**<br/> அருளில் பெரியது அகிலத்துஇல் வேண்டும் <br/> பொருளில்  தலைஇலது போல் .|
|32   |**திருவருளின் பயன்**<br/> பெருக்க நுகரவினை பேரொளியாக எங்கும் <br/> அருக்கன் என நிற்கும் அருள்.|
|33   |**அருளின்றி அறிவு பெறப்படாது**<br/> ஊன்அறியாது என்றும் உயிர்அறியாது ஒன்றும்இது <br/> தான்அறியாது ஆர்அறிவார்? தான்.|
|34   |**ஆன்மாக்கள் அருளை அறியாமைக்குக் காரணம்**<br/> பால்ஆழி மீன்ஆளும் பான்மைத்துஅருள் உயிர்கள் <br/> மால்ஆழி ஆழும் மறித்து  .|
|35   |**இதுவுமது**<br/> அணுகுதுணை அறியா ஆற்றோனில் ஐந்தின் <br/> உணர்வை உணராது உயிர்.|
|36   |**இதுவுமது**<br/> தரையை அறியாது தாமே திரிவார் <br/> புரையை உணரா புவி.|
|37   |**அருளை நாடுகிறவன் எளிதில் அறிவன்**<br/> மலைகெடுத்தோர் மண்கெடுத்தோர் வான்கெடுத்தோர் ஞானம் <br/> தலைகெடுத்தோர் தற்கேடர் தாம்.|
|38   |**இதுவுமது**<br/> வெள்ளத்துள் நாவற்றி எங்கும் விடிந்துஇருளாம் <br/> கள்ளத்து இறைவர் கடன்.|
|39   |**அருளை அறியும் வழி**<br/> பரப்புஅமைந்து கேண்மின் இதுபாற்கலன்மேல் பூஞை <br/> கரப்புஅருந்த நாடும் கடன்.|
|40   |**திருவருளை அறியாதார் சிறுமை**<br/> இற்றைவரை இயைந்தும் ஏதும்பழக்கம் இலா <br/> வெற்றுஉயிர்க்கு வீடு மிகை.|

5.  அருளுருநிலை

|||
|---|:-------------|
|41   |**குருவடிவாய் வருவது திருவருள்**<br/> அறியாமை உள்நின்று அளித்ததே காணும் <br/> குறிஆகி நீங்காத கோ.|
|42   |**அருளே ஆணவத்தைப் போக்கத்தக்கது**<br/> அகத்துஉறுநோய்க்கு உள்ளினர் அன்றி அதனைச் <br/> சகத்தவரும் காண்பரோ தான்.|
|43   |**குருவைச் சிஷ்யனன்றிப் பிறரறியார்**<br/> அருளா வகையால் அருள்புரிய வந்த <br/>பொருளாஆர் அறிவா புவி?|
|44   |**பக்குவமில்லாதார் அறியாமைக்குக் காரணம்**<br/> பொய்இருண்ட சிந்தைப் பொறிஇலார் போதமாம் <br/> மெய்இரண்டும் காணார் மிக.|
|45   |**சகலருக்குக் குருவாய் வருதலின் நோக்கம்**<br/> பார்வைஎன  மாக்களை முன்பற்றிப் பிடித்தற்காம் <br/> போர்வைஎனக் காணா புவி.|
|46   |**குரு சாத்திர தீட்சைக்கு இன்றியமையாதவர்**<br/> எமக்குஎன்? எவனுக்கு எவைதெரியும்? அவ்வ <br/> தமக்குஅவனை வேண்டத் தவிர்.|
|47   |**நயன தீட்சை**<br/> விடம்நகுலம் மேவினும் மெய்ப்பாவகனின் மீளும்<br/> கடனில் இருள்போவது இவன்கண்.|
|48   |**மூவகை ஆன்மாக்களுக்கும் அருள்செய்யும் முறை**<br/> அகலத்துஅரும் அருளை ஆக்கும் வினைநீக்கும் <br/> சகலர்க்குவந்து அருளும் தான்.|
|49   |**சிவபெருமானே குருவாதல் வேண்டும்**<br/> ஆர்அறிவார்? எல்லாம் அகன்ற நெறிஅருளும் <br/> பேரறிவான் வாராத பின்.|
|50   |**இதுவுமது**<br/> ஞானம் இவன்ஒழிய நண்ணியிடும் நற்கல்அனல் <br/> பானு ஒழியப் படின்.|

6.  அறியுநெறி

|||
|---|:-------------|
|51   |**இருவினை ஒப்பும் சத்தி நிபாதமும்**<br/> நீடும் இருவினைகள் நேர்ஆக நேர்ஆதல்<br/>கூடும் இறைசத்தி கொளல்.|
|52   |**ஆறு பொருள்கள் அநாதி**<br/>ஏகன்அநேகன் இருள்கருமம் மாயை இரண்டு<br/>ஆகஇவை ஆறு ஆதிஇல்.|
|53   |**ஆறு பொருள்களுக்கும் உள்ள தொடர்பு**<br/> செய்வானும் செய்வினையும் சேர்பயனும் சேர்ப்பவனும் <br/> உய்வான்உளன் என்று உணர்.|
|54   |**கடவுள் ஆன்மாவுக்கு உயிராய் நிற்றல்**<br/> ஊன்உயிரால் வாழும்ஒருமைத்தே ஊனொடு உயிர் <br/> தான்உணர்வோடு ஒன்றாம் தரம்.|
|55   |**கடவுள் உயிருக்கு உயிராய் நின்று நடத்தும் முறைமை**<br/> தன்னிறமும் பன்னிறமும் தானாம்கல்தன்மை தரும் <br/> பொன்னிறம்போல் மன்னிறம் இப்பூ.|

|59   |**ஞானத்தை அணையும் முறைமை**<br/> களியே மிகுபுலனாக் கருதி ஞான <br/> ஒளியே ஒளியா ஒளி.|
|60   |**பிறபொருளைக் கருதாமல் ஞானவசமாய் நிற்றல்**<br/> கண்டபடியே கண்டு காணாமை காணாமை <br/> கொண்டபடியே கொண்டு இரு.|

7.  உயிர் விளக்கம்

|---|:-------------|
|61   |**அருளை அடைதற்கு ஏது**<br/> தூநிழல் ஆர்தற்கு ஆரும்சொல்லார் தொகும்இதுபோல் <br/> தான்அதுவாய் நிற்கும் தரம்.|
|62   |**ஆன்மா அருளை அறியும் காலம்**<br/> தித்திக்கும் பால்தானும் கைக்கும் திருந்திடும்நா <br/> பித்தத்தின் தான்தவிர்ந்த பின்.|
|63   |**ஆன்மா அருளைத் தானாகக் காணமாட்டாது**<br/> காண்பான் ஒளிஇருளில் காட்டிடவும் தான்கண்ட <br/> வீண்பாவம்  எந்நாள் விழும்.|
|64   |**திருவருள் ஆணவத்தை ஒத்திருக்கும் வகை**<br/> ஒளியும்  இருளும் ஒருமைத்துப் பன்மை <br/> தெளிவு தெளியார் செயல்.|
|65   |**திருவருளை அடைந்தோர்க்குக் கடவுள் வெளிப் படுதல் எளிது**<br/> கிடைக்கத் தகுமோ நற்கேண்மையார்க்கு அல்லால் <br/> எடுத்துச் சுமப்பானை இன்று.|
|66   |**திருவருளைச் சாராதார் கடவுளை அடையார்**<br/> வஞ்சமுடன் ஒருவன் வைத்தநிதி கவரத் <br/>துஞ்சினனோ போயினனோ சொல்?|
|67   |**அருள் ஆன்மாவை வசப்படுத்தல்**<br/>தனக்கு நிழல்இன்றா ஒளிகவரும் தம்பம்<br/>எனக் கவரநில்லாது இருள்.|
|68   |**அருளை முன்னிட்டு அதற்குப் பின்னே ஆன்மா நிற்றல் வேண்டும்**<br/>உற்கைதரும் பொற்கை உடையவர்போல் உண்மைப்பின்<br/>நிற்க அருளார் நிலை.|
|69   |**அருளுக்கு முன்னே நிற்கும்போது சிவத்தைக் காணாமைக்கு ஏது**<br/>ஐம்புலனால் தாம்கண்டு அகன்றால் அதுஒழிய<br/>ஐம்புலனார் தாம்ஆர் அதற்கு.|
|70   |**இதுவுமது** <br/>தாமேதரும் அவரைத் தம்வலியினாற் கருதல்<br/>ஆமேஇவன் ஆர் அதற்கு.|

8.  ஐந்தெழுத்து அருள் நிலை

|||
|---|:-------------|
|71   | **பேரின்பத்தைப் பெறுவோரும் பெறாதோரும்**<br/>இன்புறுவார் துன்பார் இருளின் எழும்சுடரின்<br/>பின்புகுவார் முன்புகுவார் பின்.|
|72   | **இன்பத்துக்குச் சிவானுபவம் வேண்டும்**<br/>இருவர் மடந்தையருக்கு என்பயன் இன்புண்டாம்<br/>ஒருவன் ஒருத்தி உறின்.|
|73   | **அத்துவிதக் கலப்பால் பேரின்பத்தைப் பெறுவது ஆன்மாவே, சிவம் பெறுவதில்லை**<br/>இன்பதனை எய்துவார்க்கு ஈயும்அவர்க்கு உருவம்<br/>இன்பகனம் ஆதலினால் இல்.|
|74   | **முத்திநிலை அத்துவிதம் என்பது**<br/>தாடலைபோல் கூடியவை தான்நிகழா வேற்றின்பக்<br/>கூடலைநீ யோகம்எனக் கொள்.|
|75   | **முத்திநிலையை அத்துவிதம் என்றற்குக் காரணம்**<br/>ஒன்றாலும் ஒன்றாது இரண்டாலும் ஓசையெழாது<br/>என்றால் ஒன்றுஅன்று இரண்டும்இல்.|
|76   | **இனிப் பிறவி யெடாதார் வகை**<br/>உற்றாரும் பெற்றாரும் ஓவாது உரைஒழியப்<br/>பற்றாரும் அற்றார் பவம்.|
|77   | **சமாதிநிலை பலிதமாகும் முறைமை**<br/>பேய்ஒன்றும் தன்மை பிறக்கும் அளவுமே<br/>நீயொன்றும் செய்யாமல் நில்.|
|78   | **சமய ஆசாரங்களும், அவைநீங்கும் முறைமையும்**<br/>ஒண்பொருட்கண் உற்றார்க்கு உறுபயனே அல்லாது<br/>கண்படுப்போர் கைப்பொருள்போல் காண்.|
|79   | **முத்தியின்பம் இப்படிப்பட்டது என்று சொல்லத்தக்கது அன்று**<br/>மூன்றாய தன்மைஅவர் தம்மின் மிகமுயங்கித்<br/>தோன்றாத இன்பம்அது என்சொல்.|
|80   | **சீவன்முத்தி நிலை பேரன்பினால் அடையத்தக்கது**<br/>இன்பில் இனிதென்றல் இன்புஉண்டேல் இன்றுஉண்டாம்<br/>அன்பு நிலையே அது.|

|81   | **வேத ஆகமங்களில் உள்ள உண்மைகள் ஐந்தெழுத்தில் அடங்கியன**<br/>அருள்நூலும் ஆரணமும் அல்லாதும் ஐந்தின்<br/>பொருள்நூல் தெரியப் புகும்.|
|82   | **பிரணவத்துள் அடங்கும் பொருள்கள்**<br/>இறைசத்தி பாசம் எழில்மாயை ஆவி<br/>உறநிற்கும் ஓங்காரத்து உள்.|
|83   | **ஐந்தெழுத்துள் அடங்கும் பொருள்கள்**<br/>ஊன நடனம் ஒருபால் ஒருபாலாம்<br/>ஞானநடம் தான்நடுவே நாடு.|
|84   | **ஆன்மாவின் பெத்தநிலையும் முத்திநிலையும் ஐந்தெழுத்தில் அடங்குதல்**<br/>விரியம நமேவியவ்வை மீளவிடா சித்தம்<br/>பெரியவினை தீரப் பெறும்.|
|85   | **நகாராதி பஞ்சாட்சரம் உபாய மார்க்கர்க்கு உரியது**<br/>மாலார் திரோதம் மலம்முதலா மாறுமோ<br/>மேல்ஆசு மீளா விடின்.|
|86   | **நகாராதி பஞ்சாட்சரம் மோட்ச ஆசையுடையார் ஒதல்ஆகாது**<br/>ஆராதி ஆதாரம் அந்தோ அதுமீண்டு<br/>பாராது மேல்ஓதும் பற்று.|
|87   | **சிகாராதி பஞ்சாட்சரமே மோட்ச ஆசையுடையார்க் குரியது**<br/>சிவமுதலே ஆமாறு சேருமேல் தீரும்<br/>பவம்இது நீஓதும் படி.|
|88   | **வகாரம் ஆன்மாவைச் சிவமாக்குதல்**<br/>வாசிஅருளி அதைவாழ்விக்கும் மற்று அதுவே<br/>ஆசுஇல் உருவமும்ஆம் அங்கு.|
|89   | **முத்திநிலையை விளக்கும் ஐந்தெழுத்து**<br/>ஆசுஇல்நவா நாப்பண் அடையாது அருளினால்<br/>வாசிஇடை நிற்கை வழக்கு.|
|90   | **சாதனங்களுக்கு உயிராயுள்ளது திருவருள்**<br/>எல்லா வகையும் இயம்பும் இவன்அகன்று<br/>நில்லா வகையை நினைந்து.|
